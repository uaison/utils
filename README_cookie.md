# cookie
## getCookie(name:string)
根据 `name` 获取cookie

参数名|描述
|---|---|
`name`|cookie名称

## setCookie(name:string, value:any, time:string)
设置cookie

序号|参数名|描述
|---|---|---|
1|`name`|cookie名称
2|`value`|cookie值
3|`time`|cookie过期时间，时间数值加单位。单位（d:天、h:时、m:分、s:秒）

## delCookie(name:string)
根据 `name` 删除cookie

参数名|描述
|---|---|
`name`|cookie名称
